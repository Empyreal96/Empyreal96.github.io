postmarketOS for Xperia M5 (Single Sim)

So I delved into postmarketOS and managed to get a basic port for the Sony Xperia M5 (Sony-Hollyss)! (And it got merged into the main project!)

pmOS Wiki (Xperia M5) postmarketOS Homepage
What Works At The Moment:
- Touchscreen/Display
- Charging the Battery
- X11
- (In theory) USB Networking
- Flashing with pmbootstrap or standard adb sideloading.
---
I will continue working on this but as it's just me at the moment it will take a while.

Flashing to device

I advice to install through pmbootstrap due to being able to customise what packages are installed, kernel config and user name and password. 
**NOTE: I forgot to add 'msm-fb-refresher' to required packages, install it as an additional package during 'pmbootstrap init'
Remember:

To build the image you need to use 'pmbotstrap install --android-recovery-zip' to create the flashable zip.
To flash you need to use 'pmbootstrap flasher --method=adb sideload' with the device in TWRP ADB Sideload mode.

Extra Information

Just some info to keep in mind.

Requirements:

All that is required is an Unlocked Bootloader, TWRP Installed.

Notes on Installing (TWRP)

- Make a full wipe.
- Make sure 'System' and 'Data' Partitions are NOT mounted before flashing

Restoring back to Android

Not sure why but TWRP throws a wobbly when we have flashed pmOS, it failes to read/write to the System partition so before flashing Android Zips we need to:

'Click Wipe' - 'Click Advanced Wipe' - 'Tick System' - 'Click Repair or Change....' - 'Click Change Filesystem' - 'And select EXT4'

This will allow TWRP to modify the partition again!

USB Networking

An IP Address will be given to the phone when plugging into a PC, this is used for USB Networking.

USB Networking (SSH) Works.
