WPDevPortal - Windows Device Portal Client


An easy to use tool to let users access various features of Windows Device Portal, natively using UWP ad REST APIs.

Features:

- Install, uninstall and launch applications

- View running process

- View performance info (CPU, GPU, IO, Network etc)

- View Device Hardware paths
