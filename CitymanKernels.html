Pre-compiled Lumia 950 XL Kernels

I wouldn't have been able to compile these if it wasn't for Imbushuo with his 4.18.6 Kernel and config.
Here is some info on the Kernels I have compiled and uploaded for my experimemts

Linux:

I built Kernel 5.7 when it was the 'mainline' branch, with support for Bluetooth Networking so I could access the Internet and use apt while booted up in Debian, it's not perfect but that is all that I have added.

I did also compile WoA Projects' 'cherry-picked' 5.8 Kernel, but sadly I had issues with Bluetooth in general, with or without the updated Device tree, it could be the device tree, or the kernel config and the new msm8994 commits.. I chose not to upload this version

Android:

This is just a basic Android Kernel, using tweaked config from the Linux config.
I didn't make this with booting Android in mind though, all I wanted to do was boot TWRP on the device as J0Sh1X did.. 

I was successful, and even managed to flash Windows 10 Mobile with a TWRP Flashable Zip I made.. but that's a story for later..

I added NTFS Support to the kernel along with the Android configs

All the Kernels can be found over on my Github
