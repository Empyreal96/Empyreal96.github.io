Windows 10 Toolbox


An easy to use script to let users modify "difficult to change" settings in Windows 10.

Available Settings Overview:
- Windows Update
- Windows Search
- Cortana
- Windows Spotlight
- Services
- Telemetry
- OneDrive
- Powershell based Tweaks
- General Settings and tweaks
- Boot Settings
- Touchscreen Options

Note: This script modifies the Windows Registry so if you have doubts, create a registry backup
