PE Viewer


A simple UWP based tool to view basic information about PE files (exe, dll). It is not a priority application so expect some issues and slow progress

Requirements:
- Windows 10 Desktop 16299+
- Windows 10 Mobile 15254

Features:
- View hashes and signatures of the file
- Read Import and Export table
- View info and Download .PDB file for Microsoft signed files
- Show different image headers
